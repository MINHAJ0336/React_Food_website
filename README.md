# RestoHub - Food Delivery Platform

A complete restaurant management system built with React, TailwindCSS, and Supabase.

## Features

- User authentication (Login/Signup)
- Browse restaurants and menu items
- Add items to cart
- Place orders
- Track order status
- Admin dashboard for managing restaurants, menu, and orders
- Responsive design
